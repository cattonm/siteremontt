# Патч 3: src/components/RoomPreview3D.jsx Референс поведінки та значень: **room-prototype.html** (setMode, lerpCam, світло). ## 3.1 ACES-тонмапінг + фізичне світло На Canvas додати gl-пропси: \`\`\`jsx \`\`\` Світло перебалансувати (ACES темніший за лінійний): - ambientLight → видалити (його роль бере hemisphere) - hemisphereLight: intensity 0.65 - SunLight directional: intensity 1.35 → 2.4, mapSize 1024 → 2048 - pointLight люстри: intensity 4 → 10; спільний pointLight: 2.4 → 6 - emissiveIntensity смуг/плафонів ×1.3 (бо тонмапінг з'їдає піки) ## 3.2 surfaceFill: додати roughness \`\`\`js import { getSurfaceRoughness } from '../utils/proceduralTextures'; // у surfaceFill(): return { color: getSurfaceColor(kind), texture: getSurfaceTexture(kind, rx, ry), roughnessMap: getSurfaceRoughness(kind, rx, ry), }; \`\`\` В OutlinedSurface (three/Outlined.jsx) прокинути roughnessMap у meshStandardMaterial і ставити roughness={1}, коли карта є. У key матеріалу додати uuid карти (той самий прийом, що з map). ## 3.3 Damping В OrbitControls: \`enableDamping dampingFactor={0.08}\`. Для frameloop="demand": \`\`\`jsx const { invalidate } = useThree(); invalidate()} ... /> \`\`\` і після pointerup ще ~1с викликати invalidate через rAF (або тимчасово frameloop="always" під час взаємодії). ## 3.4 Контактні тіні під меблями Радіальний градієнт-blob (32 px canvas, чорний 0.32 → прозорий) на PlaneGeometry, rotation-x=-90°, y=0.005, під ліжком/шафою/гарнітуром. Дешевше за AO і працює з demand. Референс: функція blob() у прототипі. ## 3.5 Режим «Зайти в кімнату» Новий локальний стан \`const [view, setView] = useState('orbit')\`. Кнопки-пігулки поверх канваса (top-right, стиль .mode-btn з прототипу: активна = var(--accent), біла типографіка). Анімація камери — 900 мс easeInOutQuad лерп position і controls.target (функція lerpCam у прототипі; на час лерпа controls.enabled=false, кожен кадр invalidate()). Параметри FP: eye [W*0.78, 1.6, D*0.86], погляд на [W*0.3, 1.35, D*0.2]; controls: minDistance 0.01, maxDistance 0.05, rotateSpeed -0.35, polar 55–110°, azimuth без меж. Повернення — ті самі орбітальні межі, що зараз у CameraRig. Хотспоти в FP-режимі сховати (вони прив'язані до зовнішнього погляду). Бейдж: «Вид зсередини · тягніть, щоб озирнутись».
